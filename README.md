# How to write JS like a pro :rocket:

### always use triple equals
No exceptions
```
if (a === b) {
  // ...
}
```


### null check
Always check with triple equals

    ```
    if (x === null) {
        // ...
    }
    ```
    
### undefined check
Always check with triple equals and use `typeof` 

    ```
    if (typeof x === 'undefined') {
        // ...
    }
    ```
